# Code of Conduct

We want a calm, useful space.

## Our pledge
We treat each other with respect. We welcome contributors with different backgrounds and skill levels.

## Expected behavior
- Use kind, direct language.
- Assume good intent, ask questions before you accuse.
- Focus on the work: the list, the writing, the structure.

## Unacceptable behavior
- Harassment, threats, insults, slurs.
- Doxxing or sharing private info.
- Trolling PRs and issues.

## Enforcement
Project maintainers can edit, close, or remove content that breaks these rules.
If you see a problem, open an issue with links and context.

## Scope
This code applies in issues, pull requests, discussions, and other project spaces.
